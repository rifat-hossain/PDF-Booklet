# PDF-Booklet
You can easily create a PDF booklet that can be turned into a book after printing using this tool.
Recommended to use the coptic stitch after printing on "Booklet mode" and to spiral bind after printing on "Split mode".

## Dependancies
- PyPDF3
- Tkinter
